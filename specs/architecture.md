# Cerebra Technical Architecture v2.0

Key principles:
- Concepts are first-class entities
- Knowledge is a graph
- Mastery is event-derived
- Multi-subject support
- Graph data separated from graph layout

Core entities: subjects, concepts, concept_relationships, recall_events, problem_attempts, explanation_events, concept_metrics, review_schedule, graph_layouts.

Frontend: React, TypeScript, Vite, Tailwind, React Flow, TanStack Query.
Backend: FastAPI, PostgreSQL, SQLAlchemy.
